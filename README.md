# ✨ To-Do List

Uma lista de tarefas simples, bonita e funcional para organizar o dia a dia sem complicação.

---

## 💡 Sobre

Criei esse projeto para praticar **JavaScript** e fortalecer minha base em front-end.

A proposta foi ir além de uma To-Do List básica, adicionando funcionalidades que realmente fazem diferença no uso cotidiano, com foco em **usabilidade e experiência do usuário**.

---

## 🚀 Funcionalidades

* ✍️ Adicionar, editar e excluir tarefas
* ✅ Marcar tarefas como concluídas
* 📂 Organização por categorias
* 🔎 Filtro de tarefas (pendentes e concluídas)
* 📅 Definição de datas
* 🌙 Alternância entre modo claro e escuro
* 🎨 Personalização de cores
* 💾 Persistência de dados com **localStorage**

---

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript

---

## ▶️ Como executar

```bash
# Basta abrir o arquivo no navegador
tarefas.html
```
